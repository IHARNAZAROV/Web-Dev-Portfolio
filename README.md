## Run Locally

Clone the project

```bash
  git clone https://github.com/IHARNAZAROV/Web-Dev-Portfolio.git
```

Go to the project directory

```bash
  cd Web-Dev-Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
